# Premium Glassmorphic Developer Portfolio

Welcome to my professional software engineering portfolio! This is a modern, high-fidelity responsive single-page application built to showcase technical skills, full-stack projects, education history, certificates, and achievements.

## 🌌 Core Features

- **Dynamic Interactive UI:** Modern visual aesthetics with a custom-engineered WebGL canvas particle background.
- **2x2 Project Grid & Detail Modals:** Home screen highlights 4 major engineering systems in clean pairs. Clicking any project card triggers a custom blurred glassmorphic overlay highlighting the **Problem Statement** (The Challenge) and **Technical Overview**.
- **All-Projects Hub (`projects.html`):** A dedicated secondary page displaying complete details, specifications, and links for 5+ major projects.
- **Serverless Form Delivery:** Real-time email submissions directly integrated with **FormSubmit AJAX API**, providing smooth transition states (`⚡ Sending Message...` -> `✓ Message Sent!`).
- **Font Awesome vector branding:** Professional vector brand icons for all GitHub, LinkedIn, and PDF actions.

---

## 🛠️ Built With

- **Structure & Layout:** HTML5 & Semantic Web Components
- **Styling:** Vanilla CSS3 (Custom Variables, Responsive flexbox/grids, transitions, glassmorphism)
- **Logic & Effects:** ES6+ Vanilla JavaScript, WebGL & Canvas API
- **Form Delivery:** FormSubmit API
- **Icons:** Font Awesome 6.4.0
